# To run the platform on AWS

- Check the file `Env/fc-server-aws.env`
- Run the script `init-aws-letsencrypt.sh`
- Run `docker-compose -f docker-compose.aws.yml up`
- Navigate to: `https://foodchain-csg.ch/`
